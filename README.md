# D.VA BOT
d.va bot is a bot for discord which is based on the hero owned by overwatch
___
### list of commands:

<br>**d.va info**      :: returns bot info.
<br>**d.va help**      :: returns list of commands
<br>**d.va ping**      :: returns pong!
<br>**d.va ask**       :: ask d.va something.
